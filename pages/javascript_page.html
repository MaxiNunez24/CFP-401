Introducción a JavaScript

1. ¿Qué es JavaScript?

JavaScript es un lenguaje de programación interpretado que se utiliza principalmente para crear contenido web interactivo. Es uno de los tres pilares fundamentales del desarrollo web, junto con HTML y CSS.

  • HTML -> Lenguaje de marcado -> Marcado
  • CSS -> Lenguaje de diseño -> Estilado
  • JavaScript -> Lenguaje de programación -> Interacción

2. Tipos de Datos en JavaScript

JavaScript tiene varios tipos de datos que se pueden utilizar para almacenar y manipular información:

  • Number: Números, tanto enteros como decimales. Ejemplo: 42, 3.14
  • String: Cadenas de texto. Ejemplo: "Hola, mundo!"
  • Boolean: Valores de verdad. Ejemplo: true, false
  • Array: Listas ordenadas de valores. Ejemplo: [1, 2, 3]
  • Object: Colecciones de pares clave-valor. Ejemplo: { nombre: "Juan", edad: 30 }
  • Null: Representa la ausencia intencional de un valor. Ejemplo: null
  • Undefined: Representa una variable que ha sido declarada pero no inicializada. Ejemplo: undefined

3. Variables

En JavaScript, se pueden declarar variables utilizando las palabras clave var, let y const: 

// Declaración de variables
var nombre = "Juan";
let edad = 30;
const PI = 3.14;

4. Operadores

JavaScript tiene varios operadores que se pueden utilizar para realizar operaciones en variables y valores:

  • Operadores aritméticos: +, -, *, /, %
  • Operadores de asignación: =, +=, -=, *=, /=
  • Operadores de comparación: ==, ===, !=, !==, >, <, >=, <=
  • Operadores lógicos: &&, ||, !

5. Estructuras de Control

JavaScript tiene varias estructuras de control que se pueden utilizar para controlar el flujo de ejecución del código:

5.1. Condicionales

// Condicional if
if (edad >= 18) {
    console.log("Eres mayor de edad");
} else {
    console.log("Eres menor de edad");
}

5.2. Bucles

// Bucle for
for (let i = 0; i < 5; i++) {
    console.log(i);
}

// Bucle while
let i = 0;
while (i < 5) {
    console.log(i);
    i++;
}

6. Funciones

Las funciones son bloques de código reutilizables que realizan una tarea específica. En JavaScript, se pueden declarar funciones de varias maneras:

6.1. Funciones Declarativas

// Función declarativa
function saludar(nombre) {
    return "Hola, " + nombre;
}
console.log(saludar("Juan"));

6.2. Funciones Expresivas

// Función expresiva
const saludar = function(nombre) {
    return "Hola, " + nombre;
};
console.log(saludar("Juan"));

6.3. Funciones Flecha

// Función flecha
const saludar = (nombre) => {
    return "Hola, " + nombre;
};
console.log(saludar("Juan"));

7. Promesas

Las promesas son objetos que representan la eventual finalización (o falla) de una operación asincrónica y su valor resultante.

// Crear una promesa
const promesa = new Promise(
    (resolve, reject) => {
        const exito = true;
        if (exito) {
            resolve("La operación fue exitosa");
        } else {
            reject("La operación falló");
        }
    });

// Usar una promesa
promesa.then((mensaje) => {
    console.log(mensaje);
}).catch((error) => {
    console.error(error);
});
