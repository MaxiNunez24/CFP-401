Proyectos de HTML

1. Estructura Básica de una Página Web

Este proyecto consiste en crear la estructura básica de una página web utilizando HTML.

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mi Primera Página Web</title>
</head>
<body>
    <h1>¡Hola, mundo!</h1>
    <p>Este es un párrafo en mi primera página web.</p>
</body>
</html>

2. Añadir Imágenes y Enlaces

En este proyecto, aprenderás a añadir imágenes y enlaces a tu página web.

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Imágenes y Enlaces</title>
</head>
<body>
    <h1>Imágenes y Enlaces</h1>
    <p>Aquí hay una imagen:</p>
    <img src="ruta/a/tu/imagen.jpg" alt="Descripción de la imagen">
    <p>Aquí hay un enlace a <a href="https://www.example.com">Example</a>.</p>
</body>
</html>
