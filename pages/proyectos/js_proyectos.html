Proyectos de JavaScript

1. Interactividad Básica

Este proyecto consiste en añadir interactividad básica a una página web utilizando JavaScript.

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Interactividad Básica</title>
    <link rel="stylesheet" href="../../assets/estilos.css">
    <script defer src="../../scripts/interactividad_basica.js"></script>
</head>
<body>
    <h1>Interactividad Básica</h1>
    <button onclick="mostrarMensaje()">Haz clic aquí</button>
</body>
</html>
